# Homebrew Tap for Grove

Git worktree launcher — pick a branch, open it in your IDE.

## Install

```bash
brew tap ljwh1231/grove https://github.com/ljwh1231/homebrew-grove
brew install --cask grove
```

## Update

```bash
brew upgrade --cask grove
```

## Uninstall

```bash
brew uninstall --cask grove
```
